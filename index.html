About This Project

This is a System Administrator Simulation App designed to mimic a secure remote work environment. Users (staff) log in to perform work tasks in a sandboxed environment that looks like a Windows UI, while admins monitor, log, and analyze activities in real-time with the ability to take action when suspicious behavior is detected.

Admin Features

  • Real-time user monitoring
  • File access control and management
  • Threat detection and alerts
  • Activity logging and analysis
  • System policy management

Staff Features

  • Windows-like desktop environment
  • File management and document access
  • Email and browser functionality
  • Sandboxed work environment
  • Secure remote access
